## Getting Started

First, install all dependencies:

```bash
npm install
# or
yarn
```

Second, run json server:

```bash
yarn server
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
